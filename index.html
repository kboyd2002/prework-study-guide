Prework Study Guide

HTML

  • The Head element contains information about the website
  • The Body element represents the visible content shown to the user.
  • HTML is the basic framework for all webpages
  • The Head element is never seen by an end user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) and the href is using a relative path on local files
  • a link element creates the connection to the external file.
  • When using an external style sheet, we link the file using a link element placed in the HTML's head
  • using * for an element rule declares all the elements will have the style set in the body of the css code

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git push: Transmit local branch commits to the remote repository branch/I.E. uploads your new work to the branch you are working on
  • git pull: fetch and merge any commits from the tracking remote branch
  • git add: add a file as it looks now to your next commit
  • git add -A: adds all files in local repo to next commit
